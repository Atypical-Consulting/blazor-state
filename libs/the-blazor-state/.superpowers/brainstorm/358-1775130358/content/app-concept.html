What kind of demo app?

The demo should feel like a real app that shows the problems TheBlazorState solves

A

E-Commerce Store

Product catalog, shopping cart, user preferences. Shows [Persist] for product pages surviving prerender, [Shared] for cart across pages, localStorage for user prefs.

B

Task / Project Management

Boards, lists, tasks with filters. Shows [Persist] for board state, [Shared] for selected project, localStorage for filter preferences.

C

Admin Dashboard

Analytics cards, data tables, settings. Shows [Persist] for expensive API calls (charts, reports), [Shared] for theme/sidebar state, sessionStorage for tab-specific filters.
