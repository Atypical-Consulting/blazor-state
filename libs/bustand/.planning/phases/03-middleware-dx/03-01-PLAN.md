  <action>
Create MiddlewarePipeline<TState> class with `where TState : class` constraint.

**Constructor:**
Accept `IEnumerable<IMiddleware<TState>> middleware` for sync middleware.
Store in private readonly list (preserve registration order).

**InvokeBeforeChange method:**
```csharp
public bool InvokeBeforeChange(MiddlewareContext<TState> context)
```
- Iterate middleware in registration order
- Call each `OnBeforeChange(context)`
- If any returns false, stop iteration and return false (state change blocked)
- If all return true, return true (proceed with state change)
- Wrap in try/catch per middleware - if BeforeChange throws, bubble up exception to caller (they need to know validation failed)

**InvokeAfterChange method:**
```csharp
public void InvokeAfterChange(MiddlewareContext<TState> context)
```
- Iterate middleware in registration order
- Call each `OnAfterChange(context)`
- Wrap in try/catch per middleware - if AfterChange throws, log exception (using System.Diagnostics.Debug) and continue to next middleware (don't break pipeline)

**Empty pipeline handling:**
If no middleware registered, methods should be no-ops (return true for BeforeChange).

Add internal static factory method for creating empty pipeline:
```csharp
internal static MiddlewarePipeline<TState> Empty { get; } = new([]);
```
