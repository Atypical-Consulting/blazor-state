**1. Home Page (http://localhost:5001)**
- [ ] Page loads with welcome message
- [ ] Links to all examples visible

**2. Counter - Server (http://localhost:5001/counter-server)**
- [ ] Page loads with InteractiveServer indicator
- [ ] Increment/Decrement buttons work

**3. Counter - WASM (http://localhost:5001/counter-wasm)**
- [ ] Page loads (may take a moment for WASM download)

- [ ] Refresh page - count persists

**4. Counter - Auto (http://localhost:5001/counter-auto)**
- [ ] Page loads quickly (Server first)
- [ ] Mode indicator shows "Server" initially, then "WASM" after download

**5. TodoList (http://localhost:5001/todolist)**

- [ ] Can toggle completed status
- [ ] Can delete todos
- [ ] Filter buttons work (All/Active/Completed)
- [ ] Refresh page - todos persist

- [ ] Can adjust quantities
- [ ] Total price calculates correctly
- [ ] Checkout button shows loading state
- [ ] After checkout, cart is cleared
- [ ] Close browser tab, reopen - cart is empty (SessionStorage)

**7. DevTools (http://localhost:5001/bustand-devtools)**
