Refactor ZustandStore<TState> based on CONTEXT.md decisions:

1. **Replace constructor with InitialState property:**
   - Remove `protected ZustandStore(TState initialState)` constructor
   - Add `protected abstract TState InitialState { get; }` abstract property
   - Initialize `_state` lazily from InitialState on first access
   - Add `_initialized` flag to track initialization

2. **Add Set(TState newState) overload:**
   - Complement existing `Set(Func<TState, TState> mutator)`
   - Direct state replacement: `Set(new MyState { ... })`
   - Same thread-safety (locking) as mutator version

3. **Add SetAsync() for background thread safety:**
   ```csharp
   protected async Task SetAsync(Func<TState, TState> mutator)
   {
       // Captures SynchronizationContext if available
       var context = SynchronizationContext.Current;
       lock (_lock)
       {
           _state = mutator(_state);
       }
       // Post to sync context if available, otherwise invoke directly
       if (context != null)
       {
           var tcs = new TaskCompletionSource();
           context.Post(_ => { OnStateChanged(); tcs.SetResult(); }, null);
           await tcs.Task;
       }
       else
       {
           OnStateChanged();
       }
   }
   ```
   - Also add `SetAsync(TState newState)` overload

4. **Add render loop detection:**
   - Add `private bool _isRendering;` field
   - Add `internal void BeginRender()` / `internal void EndRender()` methods
   - In both Set() methods, check `if (_isRendering) throw new RenderLoopException(GetType());`
   - Component base classes (Plan 03) will call BeginRender/EndRender around StateHasChanged

5. **Add InitializeAsync() hook:**
   ```csharp
   protected virtual Task InitializeAsync() => Task.CompletedTask;

   internal async Task EnsureInitializedAsync()
   {
       if (!_isInitializing && !IsInitialized)
       {
           _isInitializing = true;
           await InitializeAsync();
           _isInitialized = true;
           _isInitializing = false;
       }
   }
   ```
   - Track with `_isInitializing` and `_isInitialized` flags
   - Make EnsureInitializedAsync internal for DI activation

6. **Implement IsInitialized property** from IStore interface

Keep existing functionality:
