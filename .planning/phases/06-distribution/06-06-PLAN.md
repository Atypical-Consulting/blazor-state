**Pages/TodoList.razor**:
```razor
@page "/todolist"
@rendermode InteractiveAuto

@*
    TodoList Page - Intermediate Complexity Example

    KEY CONCEPTS DEMONSTRATED:
    - List management (add, remove, update items)
    - Working with ImmutableList in state
    - Derived state (FilteredItems, ActiveCount)
    - Multiple UI interactions with a single store
    - Filter state (All/Active/Completed)

    COMPONENT PATTERN:
    - Uses ZustandComponent<TStore, TState> for automatic subscriptions
    - Accesses Store.FilteredItems (derived state) for display
    - Calls Store methods for all state mutations
*@

@inherits ZustandComponent<TodoStore, TodoState>

<PageTitle>TodoList</PageTitle>

<h1>TodoList</h1>

<p class="render-mode-info">
    <strong>Render Mode:</strong> Interactive Auto
    <br />
    <small>This example demonstrates list management with ImmutableList.</small>
</p>

@* Add new todo *@
<div class="todo-input">
    <input type="text"
           @bind="newTodoText"
           @bind:event="oninput"
           @onkeypress="HandleKeyPress"
           placeholder="What needs to be done?"
           class="form-control" />
    <button class="btn btn-primary" @onclick="AddTodo" disabled="@string.IsNullOrWhiteSpace(newTodoText)">
        Add
    </button>
</div>

@* Todo list *@
<div class="todo-list">
    @if (!Store.FilteredItems.Any())
    {
        <p class="text-muted">
            @(State.Filter == TodoFilter.All ? "No todos yet. Add one above!" : "No items match this filter.")
        </p>
    }
    else
    {
        <ul class="list-group">
            @foreach (var item in Store.FilteredItems)
            {
                <li class="list-group-item d-flex align-items-center @(item.IsCompleted ? "completed" : "")">
                    @* Toggle checkbox *@
                    <input type="checkbox"
                           checked="@item.IsCompleted"
                           @onchange="() => Store.ToggleTodo(item.Id)"
                           class="form-check-input me-2" />

                    @* Todo text (editable on double-click in a real app) *@
                    <span class="flex-grow-1 @(item.IsCompleted ? "text-decoration-line-through text-muted" : "")">
                        @item.Text
                    </span>

                    @* Delete button *@
                    <button class="btn btn-sm btn-outline-danger" @onclick="() => Store.RemoveTodo(item.Id)">
                        Delete
                    </button>
                </li>
            }
        </ul>
    }
</div>

@* Filter and actions *@
<div class="todo-footer mt-3">
    <span class="text-muted">@Store.ActiveCount items left</span>

    <div class="btn-group ms-3" role="group">
        <button class="btn btn-sm @(State.Filter == TodoFilter.All ? "btn-primary" : "btn-outline-primary")"
                @onclick="() => Store.SetFilter(TodoFilter.All)">
            All
        </button>
        <button class="btn btn-sm @(State.Filter == TodoFilter.Active ? "btn-primary" : "btn-outline-primary")"
                @onclick="() => Store.SetFilter(TodoFilter.Active)">
            Active
        </button>
        <button class="btn btn-sm @(State.Filter == TodoFilter.Completed ? "btn-primary" : "btn-outline-primary")"
                @onclick="() => Store.SetFilter(TodoFilter.Completed)">
            Completed
        </button>
    </div>

    @if (State.Items.Any(i => i.IsCompleted))
    {
        <button class="btn btn-sm btn-outline-secondary ms-3" @onclick="Store.ClearCompleted">
            Clear completed
        </button>
    }
</div>

@* Mark all complete *@
@if (State.Items.Any())
{
    <div class="mt-3">
        <button class="btn btn-outline-success" @onclick="Store.CompleteAll" disabled="@Store.AllCompleted">
            Mark all complete
        </button>
    </div>
}

<div class="explanation mt-4">
    <h4>Patterns demonstrated:</h4>
    <ul>
        <li><strong>ImmutableList operations:</strong> Add(), RemoveAll(), Replace()</li>
        <li><strong>Derived state:</strong> FilteredItems, ActiveCount, AllCompleted computed on access</li>
        <li><strong>Multiple mutations:</strong> AddTodo, RemoveTodo, ToggleTodo, SetFilter</li>
        <li><strong>Persistence:</strong> Todos saved to LocalStorage - refresh to verify!</li>
    </ul>
</div>

@code {
    private string newTodoText = "";

    private void AddTodo()
    {
        if (!string.IsNullOrWhiteSpace(newTodoText))
        {
            Store.AddTodo(newTodoText);
            newTodoText = "";
        }
    }

    private void HandleKeyPress(KeyboardEventArgs e)
    {
        if (e.Key == "Enter")
        {
            AddTodo();
        }
    }
}
```

**Pages/ShoppingCart.razor**:
```razor
@page "/shopping-cart"
@rendermode InteractiveAuto

@*
    Shopping Cart Page - Advanced Complexity Example

    KEY CONCEPTS DEMONSTRATED:
    - Nested objects (CartItem contains Product)
    - ImmutableDictionary for keyed collections
    - Complex derived state (TotalPrice, TotalItemCount)
    - Async operations (CheckoutAsync)
    - Multiple computed properties

    ADVANCED PATTERNS:
    - Dictionary-based state for O(1) lookups by key
    - Loading state during async operations
    - Session storage (clears when browser closes)
*@

@inherits ZustandComponent<ShoppingCartStore, ShoppingCartState>

<PageTitle>Shopping Cart</PageTitle>

<h1>Shopping Cart</h1>

<p class="render-mode-info">
    <strong>Render Mode:</strong> Interactive Auto
    <br />
    <small>This example demonstrates nested objects and async operations.</small>
</p>

<div class="row">
    @* Product catalog *@
    <div class="col-md-8">
        <h3>Products</h3>
        <div class="row">
            @foreach (var product in SampleProducts)
            {
                <div class="col-md-6 mb-3">
                    <div class="card">
                        <div class="card-body">
                            <h5 class="card-title">@product.Name</h5>
                            <p class="card-text">
                                <span class="badge bg-secondary">@product.Category</span>
                                <br />
                                <strong>@product.Price.ToString("C")</strong>
                            </p>
                            <button class="btn btn-primary btn-sm"
                                    @onclick="() => Store.AddToCart(product)"
                                    disabled="@State.IsCheckingOut">
                                Add to Cart
                            </button>
                        </div>
                    </div>
                </div>
            }
        </div>
    </div>

    @* Shopping cart *@
    <div class="col-md-4">
        <div class="card">
            <div class="card-header">
                <h3 class="mb-0">
                    Cart
                    @if (Store.UniqueItemCount > 0)
                    {
                        <span class="badge bg-primary">@Store.TotalItemCount</span>
                    }
                </h3>
            </div>
            <div class="card-body">
                @if (Store.IsEmpty)
                {
                    <p class="text-muted">Your cart is empty.</p>
                }
                else
                {
                    <ul class="list-group list-group-flush">
                        @foreach (var item in Store.Items)
                        {
                            <li class="list-group-item">
                                <div class="d-flex justify-content-between">
                                    <span>@item.Product.Name</span>
                                    <span>@item.Total.ToString("C")</span>
                                </div>
                                <div class="d-flex align-items-center mt-2">
                                    <button class="btn btn-sm btn-outline-secondary"
                                            @onclick="() => Store.UpdateQuantity(item.Product.Id, item.Quantity - 1)"
                                            disabled="@State.IsCheckingOut">
                                        -
                                    </button>
                                    <span class="mx-2">@item.Quantity</span>
                                    <button class="btn btn-sm btn-outline-secondary"
                                            @onclick="() => Store.UpdateQuantity(item.Product.Id, item.Quantity + 1)"
                                            disabled="@State.IsCheckingOut">
                                        +
                                    </button>
                                    <button class="btn btn-sm btn-outline-danger ms-auto"
                                            @onclick="() => Store.RemoveFromCart(item.Product.Id)"
                                            disabled="@State.IsCheckingOut">
                                        Remove
                                    </button>
                                </div>
                            </li>
                        }
                    </ul>

                    <hr />

                    <div class="d-flex justify-content-between">
                        <strong>Total:</strong>
                        <strong>@Store.TotalPrice.ToString("C")</strong>
                    </div>

                    <div class="mt-3 d-grid gap-2">
                        <button class="btn btn-success"
                                @onclick="HandleCheckout"
                                disabled="@State.IsCheckingOut">
                            @if (State.IsCheckingOut)
                            {
                                <span class="spinner-border spinner-border-sm me-2"></span>
                                <span>Processing...</span>
                            }
                            else
                            {
                                <span>Checkout</span>
                            }
                        </button>
                        <button class="btn btn-outline-secondary"
                                @onclick="Store.ClearCart"
                                disabled="@State.IsCheckingOut">
                            Clear Cart
                        </button>
                    </div>
                }
            </div>
        </div>
    </div>
</div>

<div class="explanation mt-4">
    <h4>Patterns demonstrated:</h4>
    <ul>
        <li><strong>Nested records:</strong> CartItem contains Product</li>
        <li><strong>ImmutableDictionary:</strong> Items keyed by ProductId for O(1) access</li>
        <li><strong>Derived state:</strong> TotalPrice, TotalItemCount, IsEmpty computed</li>
        <li><strong>Async operations:</strong> CheckoutAsync with loading state</li>
        <li><strong>Session storage:</strong> Cart clears when browser closes (try it!)</li>
    </ul>
</div>
```
